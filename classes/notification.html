Class Notification<T>

Represents a push-based event or value that an Observable can emit. This class is particularly useful for operators that manage notifications, like {@link materialize}, {@link dematerialize}, {@link observeOn}, and others. Besides wrapping the actual delivered value, it also annotates it with metadata of, for instance, what type of push message it is (next, error, or complete).
